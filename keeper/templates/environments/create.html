Environment Guidelines
Naming Convention
  • Use lowercase, descriptive names
  • Common: production, staging, development
  • Avoid spaces and special characters
Security Best Practices
  • Production environments have additional restrictions
  • Enable sync only for intended backends
  • Test configuration in development first
Synchronization
  • Test sync configuration in development first
  • Use different Vault paths per environment
  • Monitor sync status after creation
