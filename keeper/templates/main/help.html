Getting Started
New to Keeper?

Follow these steps to get started with managing your secrets:

  1. Set up Environments: Create environments (development, staging, production) to organize your secrets
  2. Create Secrets: Add your first secret with proper tags and metadata
  3. Configure Sync: Set up synchronization with external services like AWS or Vault
  4. Manage Access: Invite team members and configure appropriate permissions

Troubleshooting
Common Issues
Sync Failed
Check your credentials and network connectivity to the external service. Verify permissions in the audit logs.
Permission Denied
Contact your administrator to review your user permissions. Some actions require higher privilege levels.
Secret Not Found
Verify the secret exists and you have access to the appropriate environment. Check if the secret was recently deleted.
Login Issues
Use the demo login system for development, or contact your administrator for database authentication setup.
